# QA Checklist — Tanseq

ئەم checklist ـە بۆ پشکنینی سیستەمی هەماهەنگییە پێش ئەوەی لە production یان کارکردنی ڕۆژانە بەکاربهێنرێت.

## 1. بنەڕەتی

- [ ] سایت بە دروستی لە موبایل دەکرێتەوە
- [ ] سایت بە دروستی لە desktop دەکرێتەوە
- [ ] فونتی کوردی بە دروستی بار دەبێت
- [ ] RTL و ڕیزبەندی خانەکان بە دروستی کار دەکات
- [ ] skip link بە keyboard کار دەکات

## 2. Login

- [ ] login ـی دیمۆ بە دروستی دەباتە قۆناغی دووەم
- [ ] زانیاری هەڵە error پیشان دەدات
- [ ] دەرچوون session پاک دەکاتەوە
- [ ] بۆ production، authentication ـی ڕاستەقینە پلاندانراوە

## 3. جۆری هەماهەنگی

- [ ] هەماهەنگی هێنانەژوورەوەی کەل و پەل
- [ ] هەماهەنگی کارکردنی کرێکاران لای هاوپەیمانان
- [ ] هەماهەنگی هێنانەژوورەوەی ئۆتۆمبێل
- [ ] هەماهەنگی بردنی ئۆتۆمبێل بۆ سێرفس
- [ ] هەماهەنگی بردنەدەرەوەی ئۆتۆمبێل بەهۆی کۆتایی هاتنی گرێبەست

## 4. زانیاری گرێبەست

- [ ] ناوی کۆمپانیا پێویستە
- [ ] ژمارەی نووسراوی گرێبەست پێویستە
- [ ] ڕێکەوتی گرێبەست نابێت داهاتوو بێت
- [ ] ڕێکەوتی هەماهەنگی نابێت پێش ئەمڕۆ بێت
- [ ] هێزی هاوپەیمانان پێویستە
- [ ] هەڵبژاردنی «هی تر» خانەی نووسین دەکاتەوە

## 5. کرێکاران

- [ ] زیادکردنی چەند کرێکار کار دەکات
- [ ] سڕینەوەی کرێکار کار دەکات
- [ ] کۆپی زانیاری کرێکاری پێشوو کار دەکات
- [ ] ناوی چوارى و نەتەوە validation هەیە
- [ ] جۆری مۆبایل validation هەیە
- [ ] باج/Badge Card: بەڵێ/نەخێر کار دەکات
- [ ] باجی کاتی/ساڵی تەنیا کاتێک پێویستە کە بەڵێ هەڵبژێردرابێت

## 6. ئۆتۆمبێل

- [ ] جۆری plate یەکەم کار دەکات
- [ ] جۆری plate دووەم کار دەکات
- [ ] live plate preview نوێ دەبێتەوە
- [ ] brand و model بە دروستی دەردەکەون
- [ ] «هی تر» بۆ brand/model/color خانەی نووسین دەکاتەوە
- [ ] ڕەنگی ئۆتۆمبێل پێویستە
- [ ] سێرفس: ڕێکەوتی هێنانەوە پێویستە
- [ ] کۆتایی گرێبەست: skip worker info کار دەکات

## 7. کەل و پەل

- [ ] checkbox ـەکان بە دروستی دیار دەبن
- [ ] هیچ item ـێک هەڵنەبژێردرابێت validation دەردەکەوێت
- [ ] «هی تر» خانەی نووسین دەکاتەوە

## 8. Preview / Copy / WhatsApp / Print

- [ ] پێشنمایش زیندوو نوێ دەبێتەوە
- [ ] copy button کار دەکات
- [ ] fallback copy لە browser ـی کۆن کار دەکات
- [ ] WhatsApp URL دەکرێتەوە
- [ ] print preview دەکرێتەوە
- [ ] چاپکردن شێوازی پاک و فەرمی هەیە

## 9. Draft

- [ ] پڕکردنەوەی فۆڕم خۆکار هەڵدەگیرێت
- [ ] refresh ـی page draft دەگەڕێنێتەوە
- [ ] clear saved draft کار دەکات
- [ ] پاککردنەوەی فۆڕم warning دەدات

## 10. Accessibility

- [ ] keyboard navigation کار دەکات
- [ ] focus ring دیارە
- [ ] error summary بە screen reader role هەیە
- [ ] progress bar aria-valuenow نوێ دەبێتەوە
- [ ] labels لەسەر هەموو input ـە پێویستەکان هەن
